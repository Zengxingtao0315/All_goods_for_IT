一、IEEE 802 协议族
802.3：CSMA/CD 协议、以太网
802.2：LLC 逻辑链路控制
802.1Q：TRUNK、虚拟局域网
802.1D：STP
802.1W：RSTP
802.1S：MSTP
802.1X：端口控制协议
802.1aq：最短路径桥接
802.1ak：MVRP
802.1ad：QinQ
802.11：无线局域网、2Mbps
802.11：扩频通信技术、2.4GHz、用伪随机序列对代表数据的模拟信号进行调制
802.11b：HR-DSSS 技术、2.4GHz、11Mbps
802.11a：OFDM 调制技术、5Ghz、54Mbps
802.11g：OFDM 调制技术、2.4Ghz、54Mbps
802.11n：MIMO 和 OFDM、2.5Ghz、5Ghz、600Mbps
802.11ac (wifi5)：MIMO 和 OFDM、1Gbps、3.47Gbps
802.11ax (wifi6)：MIMO 和 OFDM、1024QAM、9.6Gbps、14Gbps
802.11be (wifi7)：MIMO 和 OFDMMU-MIMO、OFDMA、4096QAM、Multi-Link、320MHz、多资源单元 (MRU)、2.4Ghz、5Ghz、6Ghz、23Gbps、30Gbps
802.15：无线个人局域网、蓝牙
802.16：无线城域网
802.3z：帧突发
帧突发是一种用于提高网络效率的技术，它允许一台设备在一次 “信道抢占” 中，连续发送多个数据帧，而不是每发送一个帧都要重新竞争信道。
